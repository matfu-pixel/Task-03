Purpose

The purpose of this service is to simplify the work with ensembles of algorithms for a person who does not know python or another programming language, but who has data on which he wants to train a Random Forest or Gradient boosting.

How to use

Data can be added using the "Add data" button, and models can be added using the "Add model" button. Both models and data can be added many times. By clicking the "Model training" button, the user can select a pair: a model and data, for further training and obtaining predictions.

The implementations of all algorithms were made independently by the author.
